# Create T3 App

This is a [T3 Stack](https://create.t3.gg/) project bootstrapped with `create-t3-app`.

## Available Scripts

In the project directory, you can run:

### `pnpm dev`

Runs the the web client and the server

### `pnpm static`

Builds the nextjs app

### `npx cap sync`

Updates the android files
